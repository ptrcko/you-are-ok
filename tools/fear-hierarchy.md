<section>
  <h2>Fear Hierarchy</h2>
  <p>This tool arranges feared situations into an order you define, creating a view of how they relate.</p>
  <p class="notice">Purpose: externalise fears without prescribing goals or progress.</p>
</section>

<section>
  <h2>What it is for</h2>
  <ul>
    <li>Listing situations or tasks that evoke fear.</li>
    <li>Organising them in a user-defined order.</li>
  </ul>
</section>

<section>
  <h2>What it is not for</h2>
  <ul>
    <li>Enforcing exposure schedules.</li>
    <li>Scoring bravery, effort, or outcomes.</li>
  </ul>
</section>

  <h2>Status</h2>
  <p>This page is a discovery placeholder. Future versions will add optional entry management consistent with the data schema.</p>
</section>
